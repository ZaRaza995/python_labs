# Лабораторная работа №7: Тестирование и стиль кода

## Цель работы

Научиться писать модульные тесты с использованием `pytest`, измерять покрытие кода и поддерживать единый стиль с помощью `black`.

## Выполненные задачи

1. **Настройка проекта**: Создан файл `pyproject.toml` с конфигурацией зависимостей и инструментов.
2. **Тестирование модуля обработки текста (`src/lib/text.py`)**:
    * Написаны тесты для функций `normalize`, `tokenize`, `count_freq`, `top_n`.
    * Использована параметризация для проверки различных входных данных.
3. **Тестирование конвертера (`src/lab05/json_csv.py`)**:
    * Написаны тесты для `json_to_csv` и `csv_to_json`.
    * Использована фикстура `tmp_path` для работы с временными файлами.
    * Проверены негативные сценарии (отсутствие файла, некорректный формат).
4. **Стиль кода**:
    * Весь проект отформатирован с помощью `black`.

## Результаты

Все тесты проходят успешно. Код соответствует стандарту оформления.
